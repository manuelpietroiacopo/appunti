$ x \in A $ → x ∈ A  
$ x \notin A $ → x ∉ A  
$ A \subset B $ → A ⊂ B  
$ A \subseteq B $ → A ⊆ B  
$ A \supset B $ → A ⊃ B  
$ A \supseteq B $ → A ⊇ B  
$ \emptyset $ → ∅  
$ \cup $ → ∪ (unione)  
$ \cap $ → ∩ (intersezione)  
$ \setminus $ → ∖ (differenza insiemistica)

$ \forall x \in A $ → ∀x ∈ A  
$ \exists x $ → ∃x  
$ \neg P $ → ¬P  
$ P \land Q $ → P ∧ Q  
$ P \lor Q $ → P ∨ Q  
$ P \Rightarrow Q $ → P ⇒ Q  
$ P \Leftrightarrow Q $ → P ⇔ Q

$ \infty $ → ∞  
$ \approx $ → ≈  
$ \propto $ → ∝  
$ \pm $ → ±  
$ \cdots $ → ⋯  
$ \ldots $ → …  
$ \dotsc $ → … (usato in contesti matematici specifici)


$ \sqrt{x} $ → √x  
$ \sqrt[n]{x} $ → ⁿ√x  
$ x^2 $ → x²  
$ x_n $ → xₙ  
$ \sum_{i=1}^{n} i $ → ∑ᵢ₌₁ⁿ i  
$ \int_a^b f(x) dx $ → ∫ₐᵇ f(x) dx


$ a + b $ → a + b  
$ a - b $ → a - b  
$ a \cdot b $ → a · b  
$ a \times b $ → a × b  
$ a \div b $ → a ÷ b  
$ \frac{a}{b} $ → 𝑎⁄𝑏
